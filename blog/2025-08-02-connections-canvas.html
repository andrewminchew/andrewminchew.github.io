Connections Canvas

I felt like you should be able to drag the tiles around the NYT Connections game and this seemed easier than getting the NYT to change their game on my account. So I explained the idea to ChatGPT (or maybe Gemini) and here we are.

Here's how to use it:

  1. Set Connections game to light mode
  2. Screenshot the connections board
  3. Crop the screenshot to only include the word tiles
  4. Upload the screenshot with the image selector above
  5. Drag the tiles around the canvas to plan your guesses. The drag behavior can be a little fiddly. If you have a suggestion for how to fix it (or any other feedback), please drop me a line.

If you're a visual learner, check back soon. I'll be adding a video link soon.

Now go forth and puzzle!
